# cozy-vistas 🌌

A hand-picked, curated collection of high-quality atmospheric, scenic, and classical oil painting landscapes. 

## Features
* **Resolution:** High-quality assets up to 4K ($3840 \times 2400$).
* **Aspect Ratio:** Predominantly **16:10**—ideal for modern high-resolution laptops and productivity displays (easily croppable to 16:9 via OS settings).
* **Themes:** 19th-century classical oil paintings, cozy pixel art workshops, serene anime-style landscapes, and clean minimalist vector vistas.

## How to Download
To grab the entire collection at once, click the green **Code** button at the top right of this page and select **Download ZIP**.
